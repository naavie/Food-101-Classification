Model Overview

This image classifier was trained on a fine-tuned ResNet50 model and is capable of predicting 50 food classes. A full list of these classes can be found here. You can also find how the model was trained in this Jupyter Notebook.

First, using a pretrained ResNet50 model, I replaced the fully-connected layer in the output layer with a new fully-connected layer. This new layer contains 50 neurons, one for each of the subclasses of food categories. Then, the weights of the fully connected layer were initialized using Glorot initialization

How it works:

Step One. First, you need to find an image belonging to one of the 50 image subclasses. Please note, this image should be in .jpg format, however it can be of any size.

Step Two. Upload your image in the LIVE DEMONSTRATION section below and the model will return an output with (hopefully) the correct class. That's it!
